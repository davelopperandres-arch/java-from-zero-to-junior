# Java From Zero to Junior 

Este repositorio contiene una serie de proyectos desarrollados en Java,
con dificultad progresiva, enfocados en la preparación para el campo laboral.

## Tecnologías
- Java
- Programación Orientada a Objetos
- Git & GitHub

## Proyectos
1. Sistema de Control de Datos Personales
2. Sistema de Nómina Básica (próximamente)
3. Gestión de Usuarios (POO)
4. Sistema de Inventario
5. Gestión de Clientes y Ventas

## Objetivo
Desarrollar habilidades prácticas en Java aplicadas a escenarios reales
de empresa y software administrativo.
